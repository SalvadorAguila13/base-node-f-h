# Notas
```
Opciones:
      --help     Muestra ayuda                                        [booleano]
      --version  Muestra número de versión                            [booleano]
  -b, --base     Base del valor para la tabla de multiplicar[número] [requerido]
  -l, --listar   Al colocar l o listar, muestra la tabla de multiplicar en
                 consola               [booleano] [requerido] [defecto: "false"]
  -r, --rango    Al colocar r o rango, Para especificar cual es el rango en el
                 cual se hara la multiplicación           [número] [defecto: 10]
```
